# iOS Capgemini Assignments

This repository acts as a **parent repository**, designed to organize and
centralize the iOS course assignments required by Capgemini.

Each assignment is treated as an **independent project**, with its own
repository, version history, and evolution timeline. This repository exists
solely to aggregate and organize these deliveries in a single place, as
requested by the course guidelines.

---

## 📁 Repository Structure

The repository follows a simple and explicit structure:

assignments/
- assignment-i
- assignment-ii
- assignment-iii

Each folder represents a distinct course assignment.

---

## 📌 Important Note

The projects for each assignment **do not live directly in this repository**.
Each assignment is maintained as an independent project with its own source
control, preserving its complete commit history and original timeline.

This repository functions exclusively as a **logical organizer**, providing
a centralized view of all course assignments.

---

## 🚀 How to Use This Repository

This repository is **not intended for daily development**.
It exists exclusively for organization, tracking, and evaluation purposes.

Specific instructions on how to obtain, build, and run each assignment will be
documented as assignments are associated with this repository.
